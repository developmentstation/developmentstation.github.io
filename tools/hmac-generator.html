About HMAC

What is HMAC?

HMAC (Hash-based Message Authentication Code) is a mechanism for message authentication using cryptographic hash functions combined with a secret key.

Hash Algorithms

  • SHA-1: 160-bit, legacy (not recommended)
  • SHA-256: 256-bit, widely used
  • SHA-384: 384-bit, high security
  • SHA-512: 512-bit, maximum security

Use Cases

  • API authentication
  • Message integrity verification
  • Digital signatures
  • Secure communication protocols
  • Data authenticity validation

Security Notes

  • Use strong, random secret keys
  • Keep secret keys confidential
  • Use timing-safe comparison for verification
  • Consider key rotation policies
